# Task 2 - Responsive Portfolio Website with Contact Form and To-Do List

## Overview

This project was developed as part of my internship at ApexPlanet Software Pvt Ltd. The objective was to build a responsive and interactive web application using HTML, CSS, and JavaScript while applying core frontend development concepts.

## Features

### Contact Form

* User-friendly contact form interface
* Input fields for Name, Email, and Message
* Styled using CSS for a clean and modern appearance

### Form Validation

* Required field validation
* Email format verification using JavaScript
* Error messages displayed for invalid inputs

### Responsive Layout

* Navigation bar built using Flexbox
* Skills section designed using CSS Grid
* Media Queries for mobile and tablet responsiveness
* Optimized user experience across different screen sizes

### Dynamic To-Do List

* Add new tasks dynamically
* Remove completed tasks
* Real-time DOM manipulation using JavaScript
* Interactive and user-friendly interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Flexbox
* CSS Grid
* Media Queries
* DOM Manipulation

## Learning Outcomes

Through this project, I gained hands-on experience in:

* Creating responsive web layouts
* Building and styling HTML forms
* Implementing client-side form validation
* Working with Flexbox and CSS Grid
* Manipulating the DOM using JavaScript
* Handling user interactions and events
* Developing mobile-friendly web applications
